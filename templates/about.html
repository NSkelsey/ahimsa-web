The Big Picture

ahimsa enables any holder of Bitcoin to publish immutable statements within Bitcoin's block chain. These statements, which we call bulletins, are encoded within Bitcoin transactions which can be sent from a smartphone or a computer. These statements can be viewed from any computer that can connect to the Bitcoin network.

If a Bitcoin user can get a transcation into the block chain it is effectively permanent. ahimsa is based off of this principle. Any software that understands ahimsa’s messaging format and has a copy of the blockchain can then decode these bulletins and display them for the public to view.

To censor a user's bulletins, an attacker must either block access to Bitcoin completely or generate blocks faster than the entire network. Since most small nations do not have the computing infrastruture to mount a 51% attack against the Bitcoin network, ahimsa’s method for storing bulletins is robust.

In essence, we have repurposed Bitcoin as an immutable public record of free speech and we have built the tools that let anyone publish, view, and interact with these records. This system does not depend on the honesty or availability of any single website, service or hard drive. Instead, ahimsa uses the global distribution of the blockchain to ensure the availablilty of these records.

Definitions

Bulletin A message stored in the Bitcoin blockchain.

Blockchain The public ledger of every bitcoin transaction.

Topic A metadata tag used to group similar bulletins together.

Author A bitcoin address from which bulletins originate.
